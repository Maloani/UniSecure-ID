# 🔐 UniSecure-ID – Système biométrique de contrôle d’accès académique

## 📌 Présentation

**UniSecure-ID** est un système biométrique avancé conçu pour sécuriser les environnements académiques en contrôlant l’accès et en gérant les présences grâce aux technologies de **reconnaissance faciale** et **d’empreintes digitales**.

Ce système a été conçu et déployé à l’**Université de Lisala (République Démocratique du Congo)** afin de renforcer la vérification des identités, lutter contre la fraude académique et digitaliser les processus de contrôle.

---

## 🚀 Fonctionnalités principales

* 👆 **Authentification par empreintes digitales** (capteurs biométriques)
* 👁️ **Reconnaissance faciale** basée sur l’IA (OpenCV)
* 📊 **Gestion automatisée des présences** (étudiants et personnel)
* 🔐 **Sécurisation des accès** et prévention des fraudes
* 🧾 **Intégration de QR Code** pour identification rapide
* 🌐 **Tableau de bord web** pour le suivi et l’administration

---

## 🧠 Architecture du système

Le système repose sur l’intégration de plusieurs composants matériels et logiciels :

* 📡 Capteurs biométriques (empreintes digitales)
* 📷 Caméra (reconnaissance faciale)
* 🧠 Moteur d’intelligence artificielle (Python + OpenCV)
* 🗄️ Base de données (MySQL)
* 🌐 Interface web (PHP, HTML, CSS, JavaScript)
* 🔗 Microcontrôleur (Arduino)

---

## ⚙️ Technologies utilisées

### 🔹 Matériel

* Arduino

### 🔹 Logiciel

* Python (OpenCV, IA)
* PHP (Backend)
